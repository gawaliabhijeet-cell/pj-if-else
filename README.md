# Java If-Else Programs

A collection of beginner-friendly Java programs demonstrating the use of **if**, **if-else**, **if-else ladder**, **nested if-else**, and the **conditional (ternary) operator**.

These programs are useful for students learning Java programming and basic decision-making statements.

---

## 📚 Programs Included

### 1. Check Even or Odd Number
- Accept a number from the user.
- Check whether the number is even or odd using `if-else`.

### 2. Find the Largest of Two Numbers
- Accept two numbers from the user.
- Display the larger number using `if-else`.

### 3. Check Vowel or Consonant (Command Line Argument)
- Accept a character using command-line arguments.
- Check whether it is a vowel or consonant.

### 4. Check Vowel or Consonant (Scanner Input)
- Accept a character using `Scanner`.
- Determine whether it is a vowel or consonant using `if-else`.

### 5. Find the Largest of Three Numbers (If-Else Ladder)
- Accept three numbers.
- Display the largest number using an `if-else ladder`.

### 6. Find the Smallest of Three Numbers
- Accept three numbers.
- Display the smallest number using an `if-else ladder`.

### 7. Display Result Based on Marks
- Accept marks from the user.
- Display the result according to the following criteria:

| Marks | Result |
|--------|--------|
| 80 - 100 | Distinction |
| 60 - 79 | First Class |
| 40 - 59 | Second Class |
| Below 40 | Fail |

### 8. Find the Largest of Three Numbers (Nested If-Else)
- Accept three numbers.
- Display the largest number using nested `if-else`.

### 9. Find the Largest of Three Numbers (Conditional Operator)
- Accept three numbers.
- Find the largest using the ternary (`?:`) operator.
